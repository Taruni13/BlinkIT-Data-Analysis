## Project Overview
This project analyzes **BlinkIT (India's Last Minute App)** sales data to uncover insights about product performance, outlet types, and customer behavior.  
The analysis was performed using **Power BI**, with KPIs, charts, and visuals designed to provide a clear picture of sales trends and store performance.

---

## Objectives
- Understand overall **sales performance** across item types and outlet locations.  
- Identify **top-performing categories** and underperforming items.  
- Examine the relationship between **outlet size, location type, and total sales**.  
- Visualize **rating distribution** and **item visibility** patterns.  

---

## Key Insights

| Metric | Value | Description |
|--------|--------|-------------|
| **Total Sales** | 1.20M | Aggregate sales across all items |
| **Average Sales** | 140.99 | Average sales per item |
| **Average Rating** | 3.92 | Mean customer rating |
| **Sum of Item Visibility** | 563.64 | Total item visibility score |

---

## Dashboard Features
- **KPI Cards**: Show total sales, average rating, and visibility metrics.  
- **Pie Chart**: Sum of ratings by item type to evaluate customer satisfaction.  
- **Bar Chart**: Count of sales by item type highlighting best-selling products.  
- **Gauge Chart**: Count of outlet locations.  
- **Treemap**: Outlet size distribution by outlet type.

## Insights Summary

- Snack Foods and Fruits & Vegetables dominate sales volume.
- Supermarket Type 1 stores contribute the largest portion of sales.
- Higher item visibility tends to correlate with stronger sales performance.
- Smaller outlets (Type 2) show potential for optimization.
